# 🏏 IPL Power BI Dashboard
An interactive sports analytics dashboard built using IPL data (2008-2022).

## 🚀 Key Features
- **Data Modeling:** Established a 1:N relationship between Match-level and Ball-level datasets.
- **Player Analysis:** Interactive slicers for players/seasons with custom DAX measures for Strike Rate and Performance Trends.
- **Advanced Analytics:** (In Progress) Team comparisons and cumulative score progression.
